# Коммерческое предложение — Деловая игра на Битрикс24

Лендинг с коммерческим предложением на создание стендов деловой игры (ФПП, ФОП, ФЛП) на платформе Битрикс24.

**Сайт:** https://1os.github.io/FP-offer/

## Содержание

- Три направления деловой игры: производственные, офисные и логистические процессы
- Платформа Битрикс24: смарт-процессы, БП, заказы, складской учёт
- Готовые приложения: Склад сырья, Умный офис/Tuya
- Специализированные приложения: логистика, инфоцентр, управление игрой, оптимизация, кладовщик
- Интеграция с оборудованием: штрих-сканеры, принтер этикеток, IoT-кнопки

## Локальный просмотр

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

## Генерация PDF

```bash
npm install
npm run generate-pdf
```

PDF сохраняется в `docs/commercial-proposal.pdf`.

## Деплой

Сайт публикуется автоматически через GitHub Actions при push в ветку `main`.
